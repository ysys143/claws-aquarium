# [발표] 김우현 (D-Camp) - 17만원짜리 중고 PC로 AI 세컨드 브레인 만들기

## 개요
- 비개발자 PM 출신인 발표자가 17만 원짜리 중고 미니 PC와 Claude Code를 이용해 24시간 동작하는 개인 AI 세컨드 브레인 시스템을 구축한 경험을 공유한다.
- 기존 「세컨드 브레인」 책의 PARA/CODE 프레임워크를 AI 시대에 맞게 재해석하고, Obsidian·Discord·OpenClaw를 삼각 구조로 연결하여 수집·정리·증류·자기계선을 자동화했다.
- 개발 지식 없이도 Claude Hub의 기존 에이전트를 조합하는 방식으로 시스템을 구축할 수 있으며, 실제 업무(기업 정리, 프레젠테이션 제작 등)에서 체감 효과를 얻고 있다.

---

## 주요 주제/섹션

### 세컨드 브레인 개념 소개 [07:01:29]
- 디지털 도구로 생각·아이디어·정보를 외부에 저장·체계화하는 시스템
- 핵심 전제: 뇌는 기억 저장소가 아닌 사고의 엔진으로 써야 한다
- CODE 프레임워크: Capture(수집) → Organize(정리) → Distil(증류) → Express(표현)
- PARA 구조: Projects(마감 있는 진행 프로젝트) / Areas(지속 관리 영역) / Resources(참고 자료) / Archives(보관)

### AI 시대의 세컨드 브레인 재정의 [07:03:53]
- Tiago Forte의 2025년 연간 리뷰: "세컨드 브레인이라는 이름은 AI를 위해 태어난 것 같다"
- Boundary Intelligence 개념: 무엇을 아느냐보다 무엇에 접근할 수 있느냐가 지능
- AI는 자동화 도구가 아닌 사고의 파트너; 저장·연산이 싼 시대에 비싼 것은 '적시에 맞는 정보를 불러오는 것'
- 수집·분류·검색·초안 작성(CODE 전 단계)은 AI가 담당, 무엇을 저장·버릴지 판단과 방향 설정은 사람이 담당

### 구축 비용 및 하드웨어 [07:05:07]
- 11번가에서 구매한 17만 원짜리 중고 미니 PC (RAM 16GB; 실사용 기준 8GB로 충분)
- Claude Code 월 $20 구독 (현재 $100 플랜 사용)
- 전기 요금 약 월 3,000원
- 결과: TV 아래에 놓인 소형 기기 한 대로 24시간 AI 비서 운용

### 시스템 아키텍처 및 구성 [07:06:44]
- 삼각 구조: OpenClaw + Obsidian + Discord
- Discord를 통해 AI(이름: Samsanda/사만다 — 영화 「Her」의 AI 비서 이름)와 대화
- 대화 세션 전체를 Obsidian 볼트에 저장; 외부 정보 수집·에러 처리도 자동 저장
- 장기 기억 유지: 과거 세션을 요약·불러와 이어서 대화하는 커스텀 메모리 기능
- QMD 로컬 검색으로 2~3년 뒤에도 자연어 시맨틱 검색 가능
- Nextcloud + WebDAV로 멀티 디바이스 동기화 (Obsidian Sync 미사용)
- GitHub에 볼트 주기적 백업 → 마크다운 diff로 변경 이력 추적

### PARA 구조 도입과 셀프 임프루빙 [07:07:20]
- 초기에는 PARA 불사용, 이후 AI와 함께 작업하며 효율성을 직접 체감해 전면 전환
- 에이전트 정체성 파일(User Soul / Identity)도 Obsidian 볼트로 백업
- CODE에 없는 다섯 번째 요소: Self-Improving (AI 자가 발전)
  - 동일한 실수를 3회 반복한 뒤 AI가 스스로 수정하여 이후 오류 없이 동작
- 비개발자가 만들지 않아도 되는 이유: "바퀴를 재발명하지 말라" — Claude Hub에 셀프 임프루빙 에이전트가 이미 존재

### 실제 업무 활용 사례 [07:09:46]
- D-Camp 배치(1년 코호트 성장 프로그램) 기업 정리 자동화
- 진행 프로젝트 현황 리포트 일괄 생성
- 이번 발표 PPT: 약 5일 만에 준비 (프론트엔드 디자인 스킬 + PPT 제작 스킬 두 개 조합)
- 2026년 신규 자료 리뉴얼: 팀 리드 요청 후 30분 내 완료

## 핵심 포인트 및 세부 내용
- **진입 장벽 없음**: 개발자가 아니어도 Claude Hub의 기존 에이전트를 "Obsidian에 맞게 고쳐줘"만 하면 적용 가능
- **비용 효율**: 초기 구축 17만 원 + Claude 구독 + 월 전기료 약 3,000원으로 24시간 AI 비서 확보
- **메모리 설계**: 완전한 영구 메모리는 아니지만, 과거 세션 요약 불러오기로 장기 맥락 유지
- **보안 주의**: AI가 대화 중 비밀번호 등 민감 정보를 그대로 노출할 수 있으므로 '입단속' 필요 (발표자 본인도 아직 미해결)
- **MCP·RAG 등 추가 연결**: 발표자는 시간 가중치(최신 정보 우선) 방식보다 오래된 정보도 동등하게 불러오는 방식을 선호; 외부 연결은 Gmail·Google 캘린더 정도로 보수적으로 유지
- **GitHub diff 활용**: 마크다운 볼트의 변경 이력을 개발자처럼 초록·빨간 diff로 확인 가능

## 결론 및 권장사항
- 비개발자도 중고 미니 PC와 Claude Code, Obsidian, Discord만으로 실용적인 AI 세컨드 브레인을 구축할 수 있으며, 핵심은 '직접 만들려 하지 않고 이미 있는 것을 조합하는 것'이다.
- AI 세컨드 브레인의 진짜 가치는 시스템을 써 보면서 체감되므로, 완벽한 설계보다 일단 시작해 점진적으로 개선하는 접근을 권장한다.
