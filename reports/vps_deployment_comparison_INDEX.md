## 즉시 답변이 필요한 질문별 섹션

### Q: 어떤 플랫폼이 가장 저렴한가?
**→ 섹션 1.2 (월 비용)**
- 영구 무료: GCP e2-micro (월 745시간) = Oracle Free Tier
- 가장 저렴한 유료: Hetzner CX11 (€2.99/월 ~$3.25)
- 프리 티어 기간 있음: AWS EC2, Azure (12개월)

### Q: 어떤 플랫폼이 배포가 가장 간단한가?
**→ 섹션 1.1 (배포 복잡도)**
- 최고: AWS Lightsail (2-3 명령어, 5분)
- 2순위: Hetzner Terraform (3-5 명령어, 8-10분)
- 3순위: Oracle Free Tier (4-6 명령어, 10-12분)

### Q: 어디가 가장 안전한가?
**→ 섹션 1.3 (보안)**
- Tier 1 (엔터프라이즈급): AWS EC2 + GCP
- Tier 2 (중규모): Hetzner + Oracle
- 권장: AWS EC2 + IronClaw (WASM 샌드박스)

### Q: Telegram/WhatsApp 봇 호스팅에 최고는?
**→ 섹션 1.5 + 3 (메신저 적합성 + 최적 조합)**
- **Telegram 개인**: GCP e2-micro ($0)
- **Telegram 상업**: Hetzner CX11 (€2.99)
- **WhatsApp**: AWS EC2 t3.small (프리 12개월)

### Q: 스케일링이 필요하면?
**→ 섹션 1.4 (확장성)**
- 자동 스케일링: GCP > AWS EC2 > Azure
- 수동 스케일링: Hetzner (비용 제어 가능)
- 권장: GCP (1-2분 자동 확장)

### Q: 모니터링/운영이 쉬운 곳?
**→ 섹션 1.6 (운영 난이도)**
- 최고: GCP (10초 자동 모니터링)
- 2순위: AWS Lightsail (기본 모니터링)
- 3순위: Azure Monitor
- 어려움: Hetzner (자동화 필요)

| 구분 | 내용 |
|------|------|

| 추천 | 중규모, WhatsApp 초기 |
| 비용 | $0 (12개월), 이후 $5-15/월 |

| 구분 | 내용 |

| 추천 | Telegram 봇, 스타트업 |

## 각 플랫폼의 특징 (섹션 2)

### 배포 난이도 순서
1. **AWS Lightsail** (가장 쉬움) - 웹 UI만으로 가능
2. **Hetzner** - Terraform 자동화
3. **Oracle** - 웹 콘솔, 약간의 CLI
4. **AWS EC2** - CLI/Terraform 필수, 보안 설정 복잡
5. **GCP** - gcloud CLI, 프로젝트 관리
6. **Azure** - Resource Group, VNet 개념 학습 필요

### 비용 안정성 순서
1. **Hetzner** - 정액제, 변수 없음
2. **AWS Lightsail** - 정액제, 오버플로우 없음
3. **AWS EC2** - 프리 12개월 후 조심 필요
4. **Azure** - 프리 12개월 후 조심 필요
5. **GCP** - 프리 인스턴스 1개만, 추가 인스턴스는 유료
6. **Oracle** - 영구 무료이지만 성능 제한

### 자동화 가능성 순서
1. **GCP** - gcloud 명령어 최고 수준
2. **AWS EC2** - CloudFormation/Terraform 완벽
3. **Hetzner** - Terraform 완벽 지원
4. **Azure** - ARM Template/Terraform 지원
5. **AWS Lightsail** - 자동화 제한적
6. **Oracle** - 자동화 기본적
