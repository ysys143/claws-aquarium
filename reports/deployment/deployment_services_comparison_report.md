# 1-Click 배포 서비스 비교: OpenClaw 에이전트 호스팅

**작성일**: 2026-03-14
**대상**: Tencent Lighthouse, Hostinger, DigitalOcean, Cloudtype, Railway, Render, Northflank

---

## 요약

7가지 클라우드 배포 서비스를 OpenClaw 에이전트 호스팅 맥락에서 비교했습니다. **Railway**와 **Render**가 가장 간편한 1-Click 배포를 제공하며, **DigitalOcean**과 **Hostinger**는 저비용 VPS 옵션입니다. **Tencent Lighthouse**, **Cloudtype**, **Northflank**는 각각 지역별, 한국 특화, 엔터프라이즈 용도로 최적화되어 있습니다.

핵심 결론:
- **최소 비용**: Hostinger €3.79/월, DigitalOcean $6/월
- **최고 편의성**: Railway, Render (1-Click 템플릿 기반)
- **CLI 접근성**: DigitalOcean, Hostinger (full SSH), Railway, Render (제한적)
- **메시징 통합**: 모든 플랫폼에서 Telegram/WhatsApp 가능 (설정 난이도는 다름)

## OCI Free Tier & Tailscale 통합 (2026-03 meetup 추가)

### Oracle Cloud Infrastructure (OCI) Free Tier — 에이전트 게이트웨이로 활용

**소스**: 최재훈 발표 (03_최재훈_Remote_OpenClaw_다중디바이스제어)

| 항목 | 사양 |
|------|------|
| **CPU** | 4 vCPU (Ampere ARM) |
| **RAM** | 24 GB |
| **디스크** | 200 GB |
| **아웃바운드** | 10 TB/월 |
| **비용** | 영구 무료 (Always Free) |
| **주의사항** | Oracle 재정 리스크 (기업 매각/정책 변경 시 무료 정책 변경 가능) |

**인스턴스 생성 팁**:
- CPU 수 증가 옵션이 UI에 숨겨져 있음 → 화살표 클릭으로 확장 필요
- 즉시 할당 안 될 경우: 스크립트로 반복 시도 OR 페이즈(Pay-as-you-go) 업그레이드
- Pay-as-you-go 방식은 약 30분 내외로 훨씬 빠르게 인스턴스 할당 가능
- 비용 경고 알림($1~$10) 설정으로 실수 방지 권장

**에이전트 아키텍처 역할**: 리모트 게이트웨이(에이전트 런타임)로 활용. 각 디바이스는 OpenClaw Node로 연결.

---

### Tailscale VPN 통합 — 다중 디바이스 에이전트 네트워킹

**소스**: 최재훈 발표

**개요**: Zero Trust VPN으로 여러 디바이스(macOS/Linux/Windows)를 단일 네트워크로 묶는 방법.

- 공유기 포트 포워딩, 퍼블릭 IP 노출 없이 리모트 노드 간 통신 가능
- OCI 게이트웨이 + 맥 스튜디오 + 라즈베리파이5 + 윈도우 PC(WSL2)를 단일 VPN 망으로 구성
- 네트워크 설정 복잡도를 가장 간단하게 해소하는 방법으로 권장

**Tailscale vs 기존 방식 비교**:

| 방식 | 설정 복잡도 | 보안 | 비용 |
|------|-----------|------|------|
| 포트 포워딩 | 높음 | 낮음 | 무료 |
| VPN 서버 직접 구축 | 매우 높음 | 중간 | 서버 비용 |
| **Tailscale** | **매우 낮음** | **높음 (Zero Trust)** | **무료 (개인)** |

---

### Remote Gateway + Node 아키텍처

**소스**: 최재훈 발표

```
[OCI 리모트 게이트웨이]
        ↓ (에이전트 런타임)
    ┌───┴────┬───────────────┐
    ↓        ↓               ↓
[macOS Node] [Linux Node]  [Windows Node]
 사파리 제어  Chromium 제어  Chrome 제어
 macOS CAP   Linux CAP     WSL2 권장
```

**핵심 구분**:
- **게이트웨이**: 에이전트 런타임. 직접 `exec` 실행 불가 (리모트 환경이므로)
- **노드**: OS별 기능(Capability) 광고 + 실제 OS 명령 실행
- **Capability 차이**: macOS, Linux, Windows 각각 다름 (카메라, 스크린샷, 브라우저 제어 등)
- 정의되지 않은 기능은 `system_run`/`exec`를 allow list에 수동 추가 필요

**비교표 업데이트 (OCI + Tailscale 추가)**:

| 항목 | OCI Free Tier | Tailscale |
|------|--------------|-----------|
| **역할** | 리모트 게이트웨이 호스팅 | 디바이스 간 VPN 연결 |
| **비용** | 영구 무료 | 무료 (개인/소규모) |
| **설정 난이도** | 중간 (UI 숨김 옵션 주의) | 매우 낮음 |
| **보안** | 기본 VCN 방화벽 | Zero Trust |
| **적합 시나리오** | 다중 디바이스 중앙 게이트웨이 | 노드 간 네트워크 단순화 |
| **한계** | Oracle 재정 리스크 | Tailscale 서비스 의존 |
